BuxBack Privacy Policy

Last updated: 11 February 2026

BuxBack is a browser extension that provides cashback rewards for eligible Roblox purchases. This Privacy Policy explains what data we access, collect, and process when you use the extension.

1. Data We Collect

We collect and process the following information strictly to provide cashback, referral, withdrawal, and fraud prevention functionality:

Account Identifiers

  • Roblox User ID
  • Roblox username
  • Internal BuxBack user ID
  • A randomly generated device identifier stored locally in the browser

Purchase and Cashback Data

  • Item ID (asset, bundle, or game pass)
  • Item name and image (if available)
  • Purchase price
  • Cashback amount
  • Timestamps
  • The page URL where the purchase was recorded
  • Roblox purchase identifiers (when available)

Referral Data

  • Referral code
  • Referral reward records and status

Withdrawal Data

  • Withdrawal amount
  • Withdrawal status and timestamps
  • Game Pass ID submitted for payout

Security and Fraud Prevention

  • Hashed IP address
  • Device identifier

2. Local Storage

The extension stores certain data locally in your browser using Chrome storage, including:

  • Device ID
  • Session token
  • User identifiers
  • Cashback balance
  • Extension preferences

3. What We Do Not Collect

  • We do not collect your Roblox password.
  • We do not directly access your .ROBLOSECURITY cookie.
  • We do not sell personal data.
  • We do not use data for advertising purposes.
  • We do not use third-party analytics services.

The extension uses your existing Roblox login session only to retrieve your Roblox User ID and username via official Roblox APIs.

4. Purpose of Processing

Data is processed strictly for:

  • Cashback calculation
  • Referral rewards
  • Withdrawal processing
  • Fraud prevention and abuse detection
  • Core extension functionality

5. Data Retention

We retain account and reward-related data as long as you continue using BuxBack services or as necessary to process cashback and withdrawals.

Users may request deletion of their data at any time by contacting:

Email: buxback56@gmail.com

6. Data Sharing

BuxBack does not sell or rent personal data.

Data may be processed by infrastructure providers necessary to operate the service.
